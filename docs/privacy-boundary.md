# Privacy Boundary

This public repository is designed for employer review and portfolio proof only.

## Public

- General workflow notes for spreadsheet validation and controlled product updates.
- Public-safe examples of reference normalization and dry-run reporting.
- Sanitized snippets that show coding approach without private data.

## Private

- Production source code.
- Supplier files, inventory sheets, price lists, upload outputs, logs, and commercial rules.
- Product IDs, stock data, warranties, margins, and customer/order data.

## Rule

No public file should reveal supplier data, pricing strategy, live inventory, or direct production import behavior.
