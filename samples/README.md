# Sample Code Plan

This directory is reserved for Phase 3 public-safe snippets. Samples will demonstrate engineering approach without mirroring production source code.

## Planned Areas

- `php/reference-normalizer.php` — fictional reference cleanup and boundary-safe comparison.
- `php/import-row-validator.php` — generic row validation result object with no supplier-specific fields.
- `php/dry-run-report-builder.php` — simplified report row builder for proposed changes and anomalies.

## Sample Rules

- Use fictional brands, references, prices, and product names.
- Keep snippets short enough for review.
- Explain what each snippet demonstrates.
- Exclude production option names, endpoints, nonces, table names, filenames, and business-specific pricing rules.
- Do not include upload-ready CSV files, supplier data, customer data, logs, dumps, keys, or environment files.

## Status

No code samples are published yet. Phase 2 focuses on employer-friendly context and architecture; Phase 3 will add one to three sanitized PHP snippets.
