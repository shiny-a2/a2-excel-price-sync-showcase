# A2 Excel Price Sync Showcase

Public-safe showcase for a private WooCommerce spreadsheet price and stock synchronization plugin used for controlled operational imports.

This repository is not a source mirror. Production source, supplier sheets, upload-ready files, commercial pricing data, and operational exports stay private.

## What This Demonstrates

- Spreadsheet-to-WooCommerce operational workflow design.
- Product reference matching, validation, and import safety thinking.
- Batch processing boundaries for price and inventory changes.
- Operator-facing status/reporting patterns for long-running imports.
- Public-safe documentation of data hygiene without exposing private commercial files.

## Tech Stack

- WordPress plugin architecture
- WooCommerce product operations
- PHP
- CSV/spreadsheet handoff workflows
- Inventory and pricing operations
